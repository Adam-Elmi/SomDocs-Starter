# Configuration

SomDocs-Starter is designed to be highly configurable via a single `somdocs.config.json` file located in the root of your project.

## Basic Configuration

Here is an example structure of `somdocs.config.json`:

<Code filename="somdocs.config.json">
```json
{
  "repo_url": "https://github.com/Adam-Elmi/SomDocs-Starter",
  "colorScheme": "orange",
  "onTop": ["introduction", "installation"],
  "sidebar": {
    "collapsed": true
  }
}
```
</Code>

## Options

### <TextHighlight color="green">repo_url</TextHighlight>
*Type*: `string`
The URL of your GitHub repository. This is used to fetch metadata like contributors and last updated dates.

### <TextHighlight color="green">colorScheme</TextHighlight>
*Type*: `string`
Sets the primary theme color for the site.
*Available options*: `"rose"`, `"blue"`, `"orange"`, `"green"`, `"violet"`, `"indigo"`, `"teal"`.

### <TextHighlight color="green">onTop</TextHighlight>
*Type*: `string[]`
An array of page names (filenames without extension) that should appear at the top of the sidebar navigation, in the specified order.

<Code filename="somdocs.config.json">
```json
"onTop": ["introduction", "installation", "configuration"]
```
</Code>
