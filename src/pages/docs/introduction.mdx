# Introduction

Welcome to **SomDocs-Starter**!

This is a modern, feature-rich documentation starter template built with [Astro](https://astro.build). It is designed to be fast, accessible, and easy to customize, providing a premium reading experience for your users.

## Why SomDocs?

- **Performance First**: Built on Astro's zero-JS-by-default architecture.
- **Beautiful UI**: clean, modern design with a focus on typography and readability.
- **Dark Mode**: Fully supported dark mode with smooth transitions.
- **MDX Support**: Write your content in MDX, allowing you to use components inside your markdown.
- **Search**: Built-in search functionality to help users find what they need.
- **Configurable**: Easily customize themes, navigation, and behavior via a simple JSON config.

## Prerequisities

Before you begin, ensure you have the following installed:

- Node.js (v18.0.0 or higher)
- npm (v9.0.0 or higher)
